Hutchi Assignment – Dog Management API

A simple Go (Golang) REST API built using Gin, demonstrating clean architecture principles (controller → service → repository), environment-based configuration, validation, and a minimal static frontend.

Features

    RESTful CRUD APIs for Dog resources

    Layered architecture (Controller, Service, Repository)

    DTO-based request/response handling

    Input validation

    Environment-based configuration (.env)

    Static HTML frontend served by the Go server
    
    Request validation handled via utility validators

    Consistent error responses

    Clean separation of concerns
